# nautobot-docker-setup

This is a script to start nautobot on an instance with no configuration.

## Tested Environment

```
DISTRIB_ID=Ubuntu
DISTRIB_RELEASE=22.04
DISTRIB_CODENAME=jammy
DISTRIB_DESCRIPTION="Ubuntu 22.04 LTS"
```
